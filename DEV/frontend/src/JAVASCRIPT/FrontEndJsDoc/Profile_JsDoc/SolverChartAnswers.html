React function. Called when SolverChartAnswers is called and after the constructor. Fetchs the DataBase for the stats of the solver. If the user is a solver, the variable data will hold its information. Else, is_solver is set to 0, meaning False. If the DataBase is unreachable, an error will be logged in the console.

render() → {html}

Render function. If the user is a solver, checks if he has solved any tests. If he did then those tests are organized with a BarChart. Else, if he is not a solver, it warns the user that he must have 3 approved quizes to be a solver. BarChart: Component imported from "rechart". Uses horizontal rectangular bars with lengths proportional to the values they represent in order to sort information.
